# gqkw

#### 介绍
一款简易的文件传输App，基于AndroidAsync的使用实现文件传输功能，使用较多Lottie动画来可视化界面。欢迎大家Clone，Fork

#### 使用说明

1.安装App，确保电脑手机连接同一WiFi，或者电脑连接手机热点，点击一键连接

2.使用电脑登陆Ip地址，选择文件进行发送即可
